## Summary

 1. Java EE
 2. Apache Tomcat
 3. Spring MVC
 4. Hibernate
 5. MySQL
 6. JSP
 7. bootstrap
 
 The booksManager project provides:
 - CRUD actions on the Book object;
 - the ability to mark a book as read (markAsRead)
 - paged output of books
 - search for books by year of release
